# Eaglercraft 1.2.5

![preview](https://raw.githubusercontent.com/EaglerPorts/1.2.5/refs/heads/main/img/mainmenu.png)
:-:
This is a fully functional web port of Minecraft 1.2.5, playable in any modern web browser. <br> It currently supports only Singleplayer, with Multiplayer being worked on.

> [!WARNING]
> This is not meant to be a pirated version of Minecraft, it is simply just some patches for Minecraft 1.2.5 that allows it to be playable in the browser. If you haven't already, please go [Buy Minecraft](https://www.minecraft.net/en-us/store/minecraft-java-bedrock-edition-pc?tabs=%7B%22details%22%3A0%7D).

## Quick Start
### Play Now:

**Client:** https://eaglerports.github.io/1.2.5/

**Other Minecraft Web Ports:** https://eaglerports.github.io/

### Offline Downloads:
You can grab the latest build here from the [releases](https://github.com/EaglerPorts/1.2.5/releases) or compile it yourself.

## Compiling
Run the `CompileClient` script to build the JavaScript and WASM-GC clients. The results will be in the `client` folder.
